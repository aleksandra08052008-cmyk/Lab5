# Лабораторна робота №5
- Студен: Пономаренко Олександа
- Група: ІО-о51
- Номер в групі: 5

__Тема:__
Наслідування та поліморфізм

__Мета:__ 
Ознайомлення з механізмом наслідування та принципом
поліморфізму. Використання механізму наслідування та
принципу поліморфізму в мові Java. Здобуття навичок у
використанні механізму наслідування та принципу
поліморфізму.

## Варіант завдання
1) С13 = 5 =>
Визначити ієрархію дорогоцінного та напівкоштовного
каміння. Відібрати камені для намиста. Порахувати загальну
вагу (у каратах) і вартість намиста. Провести сортування
каміння намиста за цінністю. Знайти каміння в намисто, що
відповідає заданому діапазону
параметрів прозорості.

## Опис реалізації

- Наслідування (inheritance)
- Поліморфізм (polymorphism)
- Інкапсуляція
- Робота з колекціями (ArrayList)
- Обробка виключень (exceptions)

## Структура класів:

- "Gem" — базовий абстрактний клас
- "PreciousGem"  — дорогоцінне каміння
- "SemiPreciousGem" — напівкоштовне каміння
- "Necklace" — клас-контейнер для каменів
- "GemComparator" — сортування за цінністю
- "InvalidGemException" — виняток для некоректних даних
- "EmptyNecklaceException" — виняток для пустого намиста



## Функціонал

- Додавання каміння до намиста
- Обчислення загальної ваги (карати)
- Обчислення загальної вартості
- Сортування каміння за цінністю
- Пошук каміння за діапазоном прозорості
- Обробка виняткових ситуацій  


## Опис тестів
__testTotalWeight()__

Перевірити правильність обчислення загальної ваги намиста.

__testTotalPrice()__

Перевірити правильність обчислення загальної вартості намиста.

__testSortByValue()__

Перевірити правильність сортування каменів за цінністю.

__testEmptyNecklaceException()__

Перевірити обробку виключної ситуації для порожнього намиста.
